This section describes how one would configure an Apple Airport base station when the Airport is going to act as a gateway for the entire home with an analog modem connection.

To see how to set this up for a cable or DSL modem see the section on Airport Ethernet Configuration.

The first step is to select the Airport Utility



Hopefully your system will find the base station. If you cannot find the base station, check the Airport documentation on how to perform complete hardware resets.

If you cannot find the base station after a complete reset, recheck the installation in your computer and the software in your computer.



In a simple case, you will find a single base station - select the base station to configure it.



Base stations have a password - this keeps your neighbors from using the system and setting your parameters.

If you forget the password follow the hardware reset procedure in the Airport documentation.



This screen sets the identity of your wireless base station. The Severance_Home network name will be used to distinguish among different Airport wireless networks.



This screen is where the configuration steps differ between the analog modem configuration and the cable modem/DSL configuration.

First we indicate that the connection will be done using the Modem. All the IP address information disappears and the phone number PPP ID and PPP password are displayed instead.

As with most PPP setups you want to leave all the IP address router address and DNS address information blank and allow them to be supplied by the server. In this example the Airport is connected so the values assigned by the server are shown.



This configures the network in which the wireless devices will operate. Because this device is going to operate as the complete gateway for the home (both Ethernet and wireless), we will configure it to use Network Address Translation (NAT).

We also check the provide IP addresses on Ethernet,/b> so the Airport will distribute DHCP addresses to the LAN attached computers.

If you are in an area where other wireless equipment (802.11 compatible) , a simple approach to security is to exhaustively list the hardware addresses of the Airport adapters that this base station will service. You can find this information on the control panel of the computer with the wireless card installed.
