When the computer reboots, you will be presented with a login screen.

There are two different screens used for log-in. A "Windows" login (smaller screen with a windows logo) and a "Network" login. You need to see the network login. If you don't, go to the Network Control Panel and make Client for Microsoft Networks the default login.

Also make sure to press OK or Enter at this screen - not Escape. If you press Escape, you will not be logged in and file/print sharing will not work.

If you feel brave and want to alter the log-in process, see the Frequently Asked Questions for this tutorial.
